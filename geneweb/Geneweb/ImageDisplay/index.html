Module Geneweb.ImageDisplay

val print_image_file : Config.config -> string -> (unit, string) Stdlib.result

print_image_file conf fname send HTTP respose with content of an image file at the path fname. MIME type of an image is deducted from fname extension. Returns false if image wasn't found or couldn't be send.

val print_source : Config.config -> string -> unit

Display an image of given filename in images folder Filename may contain sub-folders, but cannot point outside images

val print : Config.config -> Gwdb.base -> unit

Searches image's filename in the environement conf.env and sends HTTP response with its content on the socket. If filename isn't presented, looks up personal image (portrait) for person mentionned in conf.env

val print_html : Config.config -> unit

Sends HTTP response with HTML page containg just image specified in arguments.

val print_placeholder_gendered_portrait : Config.config -> Gwdb.person -> int -> unit

prints html `<img>` tag of the default gendered portrait with square size size
