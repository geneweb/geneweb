Fond d'écran

  • Pour indiquer une image de fond pour la base "$o", il faut la mettre sous la forme suivante: 

         background="gwd$t{.exe}?m=IM;v=/toto.jpg"
    en remplaçant "toto.jpg" par le vrai nom de votre fichier image.

  • Mettez ce fichier image dans le répertoire 
         gw$/images

    que vous devrez éventuellement créer.
