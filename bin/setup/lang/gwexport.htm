    1/ -key : (plusieurs possibles) définit une liste d'individus
      exporte l'ascendance et la descendance des individus selon les critères qui suivent
      -a : Profondeur de l'ascendance
      -d : Profondeur de la descendance
      sans -a et -d, toute la base est exportée
      -ad : -ad n est équivalent à -a n -d n (déprécié, conservé pour compatibilité)

2/ -sn : (plusieurs possibles) définit une liste de patronymes
3/ -parentship : (plusieurs possibles) définit une liste de paires de personnes.
      exporte les personnes présentes sur le chemin entre les deux personnes d'une paire
4/ -c : Filtre appliqué après la sélection faite par le ET des trois filtres précédents
      -c -1 : Exclut toutes les personnes vivantes
      -c n : Exclut les personnes nées il y a moins de n ans
    |} %Ilang;en;{ Filter strategy : 
    1/ -key : (several possible) defines a list of persons
      export ascendance and descendance of the persons according to the following criteria
      -a : Depth of ascendants
      -d : Depth of descendants
      without -a and -d the whole base is exported
      -ad : -ad n is equivalent to -a n -d n (depreciated, kept for compatibility)
      -aws : export also siblings of persons
      -nsp : do not export spouses

3/ -parentship : (several possible) defines a list of pairs of persons.
      export the persons present on the path between the pair elements

      -c -1 : Exclude living persons
      -c n : Exclude persons born less than n years ago
    |}
