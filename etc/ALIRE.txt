GeneWeb 1.06

Pour demarrer avec GeneWeb, lancez votre navigateur Web.

Si vous avez installe GeneWeb, par exemple dans le repertoire
- /home/dupont/geneweb          sous Unix
- C:\home\dupont\geneweb"       sous Windows 95/NT
ouvrez la location:
     file:/home/dupont/geneweb/doc/index.htm

Une solution plus simple, sous Windows 95/NT, sans avoir besoin de
lancer préalablement le navigateur, est de double-cliquer sur le
fichier "index.htm" du répertoire "doc".

Suivez les instructions.

     -----------------

Voir le fichier CHANGES (en anglais) dans ce repertoire-ci pour les
changements par rapport aux versions precedentes.
