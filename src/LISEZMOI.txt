GeneWeb 7.01

Ouvrez la page START.htm

Si vous avez l'intention d'ouvrir une base, vous devez avoir démarré le
programme gwd en double cliquant sur l'icone gwd.bat (Windows),
gwd.sh (macOS ou Linux).
Si vous avez l'intention de gérer ou créer une base, vous devez avoir
démarré le programme gwsetup en double cliquant sur l'icone
gwsetup.bat (Windows), gwsetup.sh (macOS ou Linux).

Sur macOS (peut être aussi sur Linux), geneweb.command tue les versions
précédentes, ouvre les deux programmes, minimise la fenêtre correspondante
et ouvre START.htm.

Installation en mode CGI

Avant d'exécuter le script install-cgi.sh, vous devez renseigner dans ce fichier,
et dans le fichier install-cgi/gwd.cgi les informations concernant votre installation :
- WEB-ROOT : la racine web de votre serveur Apache
- MY-BASES : le dossier de vos bases
