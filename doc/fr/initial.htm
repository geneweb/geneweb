Démarrer avec GeneWeb


Vous avez deux façons de démarrer avec GeneWeb:
  • Vous pouvez utiliser le programme gwsetup. Dans ce cas, il vous suffit de lancer ce programme (si votre installation ne l'a pas déjà lancé automatiquement), puis d'ouvrir à l'adresse Web (locale) qu'il propose et vous suivez les instructions. L'utilisation est très simple et est donc préconisée aux débutants.

  • Sinon, si vous vous y connaissez, vous pouvez utiliser des commandes dans une fenêtre intéractive. Cette utilisation vous permet d'accéder à tous les détails des différentes commandes et de comprendre ce qui se passe, mais n'est pas recommandée si vous n'êtes pas spécialiste de ce genre d'approche. Cliquez ici pour démarrer par cette méthode.
