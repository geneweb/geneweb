Faites vos remarques par email à l'adresse suivante:
  • daniel.de_rauglaudre@inria.fr
S'il vous plait:
  • Précisez le numéro de version de GeneWeb que vous utilisez.
  • Précisez le système sur lequel vous tournez: Linux, Windows, MacOS X...
  • Si c'est un rapport d'anomalie, indiquez les circonstances de l'anomalie et les éventuels messages envoyés.
  • Si vous voulez contribuer par une nouvelle langue ou des modifications sur le lexique ou la page d'accueil de telle ou telle langue, le mieux est d'envoyer les fichiers "lexicon.txt" et "start.txt" complets: précisez bien de quelle version vous êtes parti. Dans les expressions du lexique, ajoutez votre langue dans l'ordre alphabétique de son abréviation (ex: "pt:" entre "no:" et "se:").
